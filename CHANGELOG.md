# Changelog
Всі помітні зміни в цьому проєкті будуть документуватися в цьому файлі відповідно до Семантичного Версіонування.

## [1.0.0] - 2025-10-29
### Added
- **MVP (Мінімально життєздатний продукт):** Реалізовано базову функціональність системи бронювання.
- feat(client): Додано логіку надсилання **email-підтвердження** після успішного бронювання (виконано з гілки `feature/email-confirmation`).
- feat(admin): Створено **базовий інтерфейс панелі адміністратора** для перегляду нових бронювань (виконано з гілки `feature/basic-admin-dashboard`).

### Changed
- Оптимізовано запит на перевірку доступності столиків для прискорення відповіді.

### Fixed
- Виправлено відображення повідомлень про помилки під час введення невірних контактних даних.

## [0.1.0] - 2025-10-15 (Початковий реліз)
### Added
- Ініціалізація проєкту та налаштування середовища розробки.
